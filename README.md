# 🧱 PluginBuilders

**PluginBuilders** is a personal GitHub repository that automates the process of building open-source Minecraft plugins from their source code.  
It helps keep a Minecraft server’s plugins up to date by compiling them automatically at scheduled intervals using GitHub Actions.

> ⚠️ **Important:** This repository does **not** own, modify, or relicense the plugins it builds.  
> Each plugin retains its **original license** and ownership.

---

## 🚀 How It Works

1. A GitHub Action workflow runs on a scheduled interval.  
2. It clones the source repositories of selected open-source plugins.  
3. It builds them using their official build tools (e.g., Gradle or Maven).  
4. The original license file from the upstream repository is bundled alongside the compiled outputs.  
5. The compiled `.jar` files (plus the license) are uploaded as build artifacts, which may be publicly visible on GitHub.

> 📝 **Note:** Since this repository is public, the build artifacts generated by GitHub Actions may be accessible to others.  
> This means distribution of those binaries is covered under each plugin’s original license.  
> PluginBuilders does not claim ownership or change the license of any plugin build.
